Parameters

Type: IRenderContext

The context that describes where the visual will be used in.

oldVisual
Type: Visual

The visual instance that had been returned the last time the CreateVisual method was called on this instance.

Return Value

Type: Visual

oldVisual, if this instance modified the visual, or a new visual that should replace the existing one in the canvas object visual tree.

Remarks
The CanvasControl uses this method to give implementations a chance to update an existing Visual that has previously been created by the same instance during a call to CreateVisual. Implementation may update the oldVisual and return that same reference, or create a new visual and return the new instance or null reference (Nothing in Visual Basic).
